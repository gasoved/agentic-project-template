# Tasks

Tasks are agent-readable instructions for discrete project phases. Execute them in order.

1. **Spec and plan** — Converse with the user to produce `spec/spec.md` and `spec/plan.md`. See `spec/GUIDE.md` for what these should contain.
2. **Scaffold** — Run `tasks/scaffold-project.md` to initialize the project skeleton, tooling, and quality guardrails.
3. **Implementation** — Build the project according to the plan. Additional task files may be created here to coordinate parallel agents or track milestones.
